# 面向对象编程(OOP)

### 目录

点击下方章节或使用左侧导航栏开始学习: 

* **[第一章: 导论](Introduction.md)**

    C++ 语言改进与特性、面向对象思想起源、首个 C++ 程序及基本输入输出。

* **[第二章: 使用对象](Using_Objects.md)**
  
     `std::string` 类、文件 I/O、内存模型（堆/栈/静态区）、指针与引用机制、`new`/`delete` 动态分配、`const` 常量及链接特性。

* **[第三章: 类](Class.md)**
  
    类与对象、抽象与封装、访问控制（public/private/protected）、构造函数与析构函数基础、声明与定义分离。

* **[第四章: 对象交互](Object_Interactive.md)**

    对象成员与交互、内联函数（inline）、头文件防卫（Header Guard）、`Stash` 与 `Stack` 经典容器设计。

* **[第六章: 拷贝构造函数](Copy_constructor.md)**

    拷贝构造定义与触发条件、按值传递与返回对象、深拷贝与浅拷贝陷阱、临时对象优化。

* **[第七章: 继承](Inheritance.md)**

    继承基本语法与 Is-A 关系、访问修饰符影响、构造与析构调用链、成员名冲突与隐藏。

* **[第七章: 多态性](Polymorphism.md)**

    向上转型与类型退化、动态绑定与虚函数、vtable 与 vptr 底层机制、对象切片陷阱、虚析构函数、协变返回类型。

* **[第八章: 运算符重载](Overloaded_Operators.md)**

    运算符重载语法规则、成员重载与非成员重载、常用运算符（赋值、输入输出、下标、自增自减）的重载实现。

* **[第十章: 异常处理](Exceptions.md)**

    异常抛出与捕获（try/catch/throw）、异常安全级别、栈展开（Stack Unwinding）、自定义异常、析构函数异常安全。

* **[第十一章: 模板](Templates.md)**

    泛型编程思想、函数模板、类模板、模板实例化与特化。

* **[第十二章: 迭代器](Iterators.md)**

    迭代器设计模式、STL 容器迭代器、迭代器分类、迭代器失效陷阱与安全实践。

* **[第十六章: 设计模式](Design.md)**

    面向对象设计原则（开闭原则等）、单例模式、工厂模式、策略模式、观察者模式等常见设计模式。

* **[设计模式案例分析](Patterns_Design-Case_Study.md)**

    基于实际案例的面向对象设计模式综合演练与代码重构。
